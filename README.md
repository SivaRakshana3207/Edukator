# 🎓 Edukator - Distance Education Platform

Edukator is a full-stack, responsive distance learning platform designed to make online education simple, interactive, and engaging. Users can explore high-quality course categories, enroll in trending courses, read insightful articles, and stay updated with the latest in personal growth and entrepreneurship.

## 🛠️ Tech Stack

**Frontend:**
* **HTML5** - Semantic structure and layout.
* **Tailwind CSS** - Modern, utility-first styling for slick UI.
* **JavaScript (ES6+)** - Dynamic DOM manipulation, testimonials slider, and API fetching.

**Backend & Database:**
* **Node.js & Express.js** - Robust server-side architecture and RESTful API endpoints.
* **SQLog** - Secure, relational database storage for users, courses, and enrollments.

---

## ✨ Features

* **Dynamic Landing Page:** Clean, modern interface designed from scratch with Tailwind CSS.
* **Course Explorer:** Categorized browsing (Personal Development, Business, Entrepreneurship, Photography).
* **Real-time Enrollment:** Displays dynamic pricing, student enrollment counts, and ratings.
* **Animated Stats Counter:** Displays live platform milestones (5000+ Involved Teachers, 1200+ Selection of Courses).
* **Dynamic Blog/News Feed:** Articles written by professional authors to keep students updated.
* **Newsletter Subscription:** Functional email updates subscription system.
* **Fully Responsive:** Seamlessly optimized for Mobile, Tablet, and Desktop screens.
